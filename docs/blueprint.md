# **App Name**: Atelier Home UK

## Core Features:

- Product Catalog: Browse and filter furniture by type, style, price, and material.
- Product Detail Pages: View detailed information, including images, descriptions, dimensions, and customer reviews.
- Shopping Cart: Add and manage items in a shopping cart, with options to update quantities and remove items.
- Secure Checkout: A streamlined and secure checkout process with multiple payment options (credit/debit cards, PayPal, etc.) and UK address validation.
- Order Tracking: Track the status of orders from placement to delivery within the UK.
- AI-Powered Style Recommendations: Suggest furniture styles based on user preferences, using a tool that determines relevant elements such as user provided photos of their home or style selections.
- Customer Support: Integrated customer support chat or contact form for inquiries and assistance.

## Style Guidelines:

- Primary color: Muted olive green (#A3B18A) to convey a sense of nature, tranquility, and connection to classic British design.
- Background color: Very light desaturated olive (#F4F5F0) provides a soft, unobtrusive backdrop.
- Accent color: Dusty rose (#C0A0A0) for calls to action and highlighting key elements, adding a touch of sophistication.
- Headline font: 'Playfair', a serif with an elegant, fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif that pairs well with 'Playfair'.
- Clean, minimalist line icons to represent furniture categories and website features.
- A grid-based layout with generous whitespace for a clean and modern feel.
- Subtle animations and transitions to enhance user experience and engagement.
